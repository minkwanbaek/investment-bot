# Vision

## Project name
Investment Bot

## Goal
Build a modular crypto investment system that can evolve from analysis and paper trading into carefully controlled live trading.

## Product stance
This project will not start as a fully autonomous profit-maximizing black box. It will begin as a disciplined investment operating system with:
- market data ingestion
- deterministic strategy execution
- explicit risk controls
- observability and journaling
- staged rollout from backtest -> paper -> limited live

## Success metrics
- Net portfolio return remains positive after fees
- Maximum drawdown stays within configured limits
- External API and infra costs remain proportionate to account size
- Every trade is explainable through strategy and risk rules
